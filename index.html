りとるまーずプロジェクト

「かみらえっと」個人により運営されている、謎のバーチャル火星探査計画です。同人サークルなのかもしれませんが同人誌を出したことはありません。
「かみらえっと」の健やかな生存のため、精神的な充足および知的好奇心を満たす事を目的として運営されてます。

汎用Misskeyインスタンス「かせいすきー」

誰でも自由に参加できる 分散型SNSです。以下の3つを目標に運営しております。

  • 1年間以上の継続した安定的な稼働
  • 1万人の移住者
  • 快適で心地よい環境づくり

安定稼働を目標に掲げてはおりますが、管理人の個人運営のため突然の死を迎える可能性があります。ご了承ください。また、参加にあたっては必ず「利用規約」をお読みください。

Minecraft マルチサーバー「りとるまーず鯖」

Minecraft Java Edition 1.19.4 を対象にした一般的なマルチサーバーです。統合版での参加は出来ません。参加には「かせいすきー」への登録が必要になります。管理人の個人運営のため、突然の死を迎える可能性があります。ご了承ください。

サーバールール等については「かせいすきー」内のチャンネルをご覧ください。
